変数

変数宣言は型注釈が無いとエラー． 

            
                let x: number;       // 数値型
                let y: boolean;      // 真偽値型
                let z: number = x + 0;   // 宣言と同時に初期化

演算子

            
                x = 1 + 1;  // xに2が代入される
                y = 1 - 1;  // xに0が代入される
                z = 2 * 3;  // xに6が代入される
                u = 6 / 2;  // xに1.5が代入される
                v = 4 % 3;  // xに1が代入される
            
        

            
                1 < 2;   // true
                1 <= 2;  // true
                1 > 2;   // false
                1 >= 2;  // false
                1 == 2;  // false
                1 != 2;  // true
                !false;  // true
                true || false // true
                true && false // false

if文

            
                let x: number = 8;
                if (x > 10) {
                    x = 0;
                } else if (x > 5) {
                    x = 1;
                } else {
                    x = 2;
                }

while文

            
                let x: boolean = true;
                while(x) {
                    x = false;
                }

for文

            
                let sum: number = 0;
                let i: number = 1;
                for(let i: number = 1; i <= 10; i = i + 1) {
                    sum = sum + i;
                }

関数

関数の定義でも入力と出力に型注釈が無いとエラー． 関数は定義するより前に使うことが出来る． 

            
                myfunc(10);

                function myfunc(i: number): number {
                    return i+1;
                }

move(θ: number): number

            
                move(45); // 45度方向にキャラクターが10px移動

fire(θ: number): number

            
                fire(45); // 45度方向に火の玉を発射

random(): number

            
                random(); // 0以上1未満の疑似乱数を生成

fork(): boolean

            
                fork(); // キャラクターが分裂する
                        // trueが返る
